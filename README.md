# Personal Finance Manager

A static personal finance manager built with plain HTML, CSS, and JavaScript.

## Run

Open `index.html` directly in a browser.

## Features

- Dashboard with income, expenses, balance, and savings
- Income and expense tracking
- Full transaction create, edit, and delete
- Transaction history with filters
- Category management
- Analytics charts
- Local AI-style financial insights
- Local storage persistence

No React, Next.js, Node.js, npm, build tools, authentication, backend, OpenAI, external APIs, or integrations are used.
